# TranscriptSummarizer
Repository to summarize a transcript by speaker name.

# Installation and use
1. Clone the repository to your local drive.
2. run "npm install" to install the required node modules.
3. run "npm run build" to create the dist folder.
4. run "npm.cmd start" to launch the application locally.

# Config
The app comes with a local app.config.json. It can be modified to work with a local llm hosted on LMStudio or connected to Gemini via api key. Other providers are not tested at the first release.
